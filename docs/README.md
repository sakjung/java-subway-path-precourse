# 기능 구현 목록

## 초기화 세팅
- 역 초기화
- 노선 초기화
- 구간 초기화 (역, 거리, 시간)

## 최소 시간 경로 조회
- 출발역과 도착역을 입력받아 경로를 조회
  - 예외 상황:
    - 등록되지 않은 출발역 혹은 도착역
    - 출발역과 도착역이 같음
    
- 출발역과 도착역사이의 가능한 노선 구하기
- 노선들의 길이 총합 구하기
- 총합들 비교 후 가장 짧은 노선 구하기
- 총 거리, 총 소요 시간, 경로 출력

## 최단 거리 경로 조회
- 출발역과 도착역을 입력받아 경로를 조회
    - 예외 상황:
        - 등록되지 않은 출발역 혹은 도착역
        - 출발역과 도착역이 같음
- 출발역과 도착역사이의 가능한 노선 구하기
- 노선들의 거리 총합이 가장 작은 노선 구하기
- 총 거리, 총 소요 시간, 경로 출력
